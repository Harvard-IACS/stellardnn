Get started

ASTROMER is a transformed-based model which was pre-trained on millions of light curves. We use ASTROMER representation to create informative light curves embeddings. the ASTROMER pre-trained encoder can be finetuned on other domains for solving downstream tasks. Read the paper for more information.

Install our python package including all the functionalities

pip install ASTROMER
